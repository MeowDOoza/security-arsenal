# 📂 Log Analysis Tools

В этом разделе собраны утилиты для парсинга и анализа системных журналов.

## 🛠 ip_extractor.py
Инструмент для быстрого извлечения IP-адресов из неструктурированных данных.

### Для чего это нужно?
При анализе инцидентов безопасности (например, атака на сервер) у тебя на руках оказываются тысячи строк логов. Этот скрипт за доли секунды вычищает мусор и дает тебе список "действующих лиц" (IP-адресов).

### Примеры использования:
- **Прямой ввод:**
  `python3 ip_extractor.py "connection failed from 192.168.0.1"`
- **Обработка лог-файла:**
  `python3 ip_extractor.py access.log`
- **Выгрузка списка для рапорта:**
  `python3 ip_extractor.py auth.log > suspects.txt`
