# 📊 Log Analysis Toolkit

Набор системных утилит для анализа и фильтрации больших текстовых данных.

---

### 1. Log Searcher (`log_searcher.py`)
**Что делает:** Сверхбыстрый поиск слов в файлах любого объема (20GB+).
**Зачем:** Для форензики и выявления ошибок в гигантских логах.
**Кратко:** Читает построчно, не ест память, игнорирует регистр.

### 2. IP Extractor (`ip_extractor.py`)
**Что делает:** Извлекает все уникальные IPv4 адреса из текстового файла.
**Зачем:** Сбор статистики посещений или формирование черных списков.
**Кратко:** Использует регулярные выражения (Regex).

---
*Подробные инструкции по работе каждого скрипта находятся внутри самих файлов в комментариях.*
