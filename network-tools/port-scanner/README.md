# 🛰️ Port Scanner (v1.0)

Профессиональный TCP сканер портов на базе стандартной библиотеки `socket`.

### Возможности:
- Сканирование конкретных IP и доменов.
- Выбор списка портов через аргументы `-p`.
- Высокая скорость за счет оптимизированного тайм-аута.

### Как запускать:
```bash
# Сканирование стандартных портов
python3 port_scanner.py 8.8.8.8

# Сканирование конкретных портов
python3 port_scanner.py google.com -p 80,443,8080
